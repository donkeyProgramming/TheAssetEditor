Camera control

All 3d editors in AssetEditor has the same camera controls. The tool supports two different controll schemas - Blender style and AssetEditor Style. AssetEditor is the default. This can be changed under settings.

Camera control - General

  • F4 - Resets camera back to start.
  • Numpad 1 / Ctrl + Numpad 1 - Set camera to front/back view
  • Numpad 3 / Ctrl + Numpad 3 - Set camera to left/right view
  • Numpad 7 / Ctrl + Numpad 7 - Set camera to Top/bottom view
  • Numpad 7 - Toggle perspective/orthographic camera
  • Numpad decimal - Focus selected object
  • Spacebar - View only selected camera

Its also possible to press the corner of the top right gizmo to move the camera into orthographic mode.

Camera control - AssetEditor Style

  • Ctrl + Mouse wheel - Camera zoom
  • Ctrl + Left Mouse button - Rotate the camera
  • Ctrl + Right Mouse button - Move the camera

Camera control - Blender Style

  • Mouse wheel - Camera zoom
  • Middle Mouse button - Rotate the camera
  • Shift + Middle Mouse button - Move the camera
