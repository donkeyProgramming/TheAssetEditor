Installing Asset Editor

AssetEditor is distributed from the GitHub releases page. Download the latest release from GitHub Releases.

Download AssetEditor_versionXX.rar from the release assets. Do not download the source code archive.

Download and extract

  1. Open the latest GitHub release page.
  2. Download AssetEditor_versionXX.rar.
  3. Extract the archive before starting the application.

Place the extracted application somewhere you have write access, for example your Desktop or inside C:\Users\YourUserName.

AssetEditor also creates a user data folder at C:\Users\YourUserName\AssetEditor. This folder is used for settings and other files created by the tool.

First startup

When AssetEditor starts for the first time, the settings window is shown automatically. Choose the game you want to work with and fill in the path to where that game is installed.

The selected game controls several important behaviors in the tool, including RMV model format version, WSModel handling, and available animation slots.

If the wrong game or installation path is selected, parts of the editor may not work as expected.
