Getting Started

What is AssetEditor

AssetEditor is a modding program for Total War games. The main focus of the tool is the Warhammer games, but a lot of the tools also work for older games.

The tool consists of the following editors

  • KitbashEditor - A simple to use 3d modelling program for editing meshes.
  • SuperView - View and edit metafiles which effects how the game interact with animations.
  • AudioTool - Edit sound events
  • SkeletonTool - Edit skeletons
  • Animation Bin - Edit or create animation sets

Configuring AssetEditor

When AssetEditor is opened for the first time, the settings window is displayed. The tool requires a game folder to be set for the selected game. If its not set, the tool will not behave correctly. The settings window can also be accessed from the tool bar. File => Settings. Some settings changes require the tool to restart.

The basics

AssetEditor works as a "super editor", with multiple different editors for different files types. The general layout is like seen below.

  • Toolbar - Tools and other functionality that works independently of selected editor
  • Editor Tabs - A list of all open tabs. Open or close as needed
  • Selected Editor - The editor surface of the selected editor
  • Editor Toolbar - The toolbar for the current editor
  • PackFile explorer - A tree view, just like windows file explorer. This allows you to view the content of selected packfiles.
