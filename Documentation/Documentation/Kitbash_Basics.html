What the Kitbash Editor Does

The Kitbash Editor inside AssetEditor is a beginner-friendly tool that lets modders create new unit appearances by combining parts from existing Total War models.

A simple way to think about it:

The Kitbash Editor lets you build a new model by mixing and matching pieces from existing game models instead of creating a model from scratch.

Examples of what you can do

  • Give a unit a different helmet
  • Swap weapons between characters
  • Attach armor from another faction
  • Combine creature and humanoid parts
  • Adjust the size or thickness of armor pieces
  • Create completely new visual variants of units

Instead of sculpting models like a professional 3D artist, you are assembling them like digital LEGO pieces.

Why the tool exists

Before the Kitbash Editor, creating new models usually required advanced software such as:

  • Blender
  • Maya
  • 3ds Max

It also required knowledge of rigging and animation systems, which can be difficult for beginners.

The Kitbash Editor was designed to make visual unit modding easier and faster.

What the Kitbash Editor helps modders achieve

  • Create new unit appearances quickly
  • Reuse existing game assets safely
  • Avoid complicated 3D modelling software
  • Test visual ideas rapidly
  • Learn modding step-by-step

Types of projects you can make

1. Variant Units

Example: the same soldier with a different helmet, shoulder armor, or weapon.

Result: a new visual version of an existing unit.

2. Custom Characters

Example: combine armor, helmets, cloaks, and weapons from multiple factions.

Result: a unique hero or lord appearance.

3. Hybrid Models

Advanced users can combine multiple creatures or armor sets into completely original designs.

Important Feature: Re-Rigging

When combining parts from different models, their skeleton structures may not match.

The Kitbash Editor includes a feature called re-rigging, which:

  • Connects meshes to skeletons
  • Prevents animation problems
  • Keeps models from stretching incorrectly

This step helps ensure the model animates correctly inside the game.

Main Goal of the Kitbash Editor

The overall purpose of the Kitbash Editor is to help modders create new visuals without needing professional 3D modelling experience.

  • Reuse existing game assets efficiently
  • Prototype ideas quickly
  • Lower the difficulty of starting modding
  • Act as a stepping stone toward advanced modelling workflows

Simple Beginner Explanation

The Kitbash Editor is a tool that lets you open Total War models and build new ones by swapping and combining parts from existing units. It works like assembling a custom miniature using pieces from different kits, but inside a digital 3D environment.
