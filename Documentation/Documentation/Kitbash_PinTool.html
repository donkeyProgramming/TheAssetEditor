Pin Tool

The Pin Tool transfers bone weight and rigging information from one or more source meshes to a set of target meshes. This is used when you have imported or created new geometry that needs to follow an animated skeleton, but does not yet have the correct bone indices or blend weights.

The tool has two modes that solve different problems:

  • Pin — attaches rigid objects to one point on the skeleton. Every vertex in the target mesh gets the same bone weights, so the mesh moves as a solid piece.
  • Skin Wrap — transfers smooth deformation weights. Each vertex in the target mesh gets bone weights interpolated from the nearest point on the source mesh surface, so the mesh deforms naturally with the skeleton.
Short version: Use Pin for rigid attachments (weapons, shields, buckles). Use Skin Wrap for deformable things (clothing, capes, skin, armor that should bend with the body).

What problems it solves

  • Giving new or imported meshes the rigging they need to follow an existing skeleton.
  • Attaching rigid accessories to a specific bone without manually setting weights.
  • Transferring smooth skin weights from an already-rigged body mesh to new clothing or armor pieces that cover the same area.
  • Quick re-rigging when source and target meshes are spatially aligned (same pose, overlapping or nearby surfaces).

Typical use cases

  • Pin: a weapon, shield, or helmet ornament that should rigidly follow one bone.
  • Pin: a simple rigid plate or buckle that attaches to the chest or belt.
  • Skin Wrap: a new shirt, cloak, or armor piece draped over a body mesh that already has good weights.
  • Skin Wrap: transferring rigging from an existing character body to replacement body parts after kitbashing.

Before you open it

  • The target meshes (meshes that will receive new rigging) should be positioned correctly relative to the source. Skin Wrap relies on spatial proximity.
  • None of the meshes should have a non-zero pivot point. The tool will warn you if this is the case.
  • For Skin Wrap, make sure the source mesh already has correct rigging and covers the area of the target mesh. Vertices far from any source surface will get weights from the nearest triangle, which may not be meaningful.
  • For Pin, you need to know which vertex on the source mesh represents the attachment point. Switch to vertex selection mode before setting the source.

How to use — Pin mode

  1. Open the Pin Tool.
  2. Set the mode to Pin.
  3. In the viewport, select the mesh(es) that should receive rigging.
  4. Press Add selected meshes to add them to the target list.
  5. Switch to vertex selection mode in the viewport.
  6. Select a vertex on the already-rigged source mesh at the point where you want the target to attach.
  7. Press Set from selected Vertex to capture the source vertex.
  8. Press Apply. Every vertex in the target meshes will receive the bone weights from that single source vertex.
Tip: Pick a vertex that is on the bone you want the rigid object to follow. If the source vertex has influence from multiple bones, the target will inherit that same mix — which is usually not what you want for a rigid attachment. Choose a vertex with a single dominant bone for a clean result.

How to use — Skin Wrap mode

  1. Open the Pin Tool.
  2. Set the mode to Skin Wrap.
  3. In the viewport, select the mesh(es) that should receive rigging.
  4. Press Add selected meshes to add them to the target list.
  5. In the viewport, select the rigged source mesh(es) you want to transfer weights from.
  6. Press Add selected meshes in the source area to add them.
  7. Press Apply. For each vertex in each target mesh, the tool finds the closest point on any source mesh surface and interpolates the bone weights from that triangle.
Multiple source meshes: You can add more than one source mesh. The tool will find the closest point across all source meshes for each target vertex. This is useful when different body parts (torso, arms, legs) are separate meshes.

What each control does

Control What it does
Animation transfer mode Switches between Pin and Skin Wrap.
Apply animation to — Add selected meshes Adds the currently selected meshes in the viewport to the target list.
Apply animation to — Remove all Clears the target mesh list.
Set from selected Vertex (Pin mode) Captures the currently selected vertex and its mesh as the rigging source.
Add selected meshes (Skin Wrap source) Adds selected viewport meshes to the source list for weight transfer.
Remove all (Skin Wrap source) Clears the source mesh list.
Apply Executes the rigging transfer and closes the window.

What to watch out for

  • Pivot points must be zero. The tool warns you if a mesh has a non-zero pivot point. Rigging transfer will produce incorrect results in that case.
  • Source and target must not overlap. A mesh cannot appear in both the target list and the source. The tool will refuse to apply if it detects this.
  • Spatial alignment matters for Skin Wrap. The tool finds the nearest point on the source surface. If the target mesh is far from the source, the weights will come from whatever triangle happens to be closest, which may not be the correct body part.
  • Vertex format changes. Both modes change the target mesh vertex format and skeleton name to match the source. This is required for the mesh to animate with the target skeleton.
  • Undo is supported. Both Pin and Skin Wrap are undoable commands. If the result is not right, undo and try again.
Important: Skin Wrap works best when the target mesh closely overlaps the source mesh in 3D space. If the meshes are far apart or in very different poses, the nearest-point lookup will produce poor results. Align the meshes before running the tool.

Pin vs Skin Wrap — when to use which

Use Pin when… Use Skin Wrap when…
The object should move rigidly (weapon, shield, ornament) The object should deform with the body (clothing, skin, flexible armor)
You want all vertices to follow the same bone(s) You want each vertex to get weights from its nearest surface point
You know exactly which bone the object should attach to You have a rigged body mesh that covers the same area as the target
Quick — just select one vertex Automatic — processes every vertex based on proximity

When it struggles

  • When the target mesh has regions far from any source mesh surface (e.g., long flowing parts with no nearby source geometry).
  • When the source mesh has very sparse triangulation in the area closest to the target.
  • When the source and target are in very different poses or positions.
  • When the mesh requires carefully hand-painted weights that cannot be approximated by proximity.
