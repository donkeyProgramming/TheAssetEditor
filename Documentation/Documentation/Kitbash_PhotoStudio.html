KitbashEditor - PhotoStudio

The Photo Studio tool is a way to create high quality images of what is currently on the screen. The tool is found under the Rendering toolbar, or by pressing Ctrl+P. The tool allows the user to save and load the current light and camera setup. To update the camera values inside the tool, press the "Refresh Camera Values" button. When "Take screenshot" button is pressed, the images is saved to disk, at the same location where AssetEditor is running. The folder is automatically opened.
